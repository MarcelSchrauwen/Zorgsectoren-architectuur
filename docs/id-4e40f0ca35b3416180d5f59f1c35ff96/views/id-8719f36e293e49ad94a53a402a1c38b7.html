06. Huisartsenzorg ()
In deze omgeving is vanuit de kapstok van referentiedomeinmodellen een architectuur uitgewerkt. Deze architectuur beschrijft interoperabiliteit use cases die van toepassing zijn binnen de sectoren en waarbij de diensten van VZVZ een ondersteunende rol (kunnen) vervullen. Voor sectoren zonder referentiemodel is de sector zelf de kapstok.
